Développement web

Le projet : Playlist Matinale

Informations

Création d'une page web en HTML CSS et Javascript comme une application de musique.

  • structurer la page web en utilisant HTML pour définir la structure du contenu
  • styliser la page avec CSS pour créer un design attrayant et responsive
  • ajouter des fonctionnalités interactives avec Javascript, telles que la lecture de musique, le contrôle du volume et la manipulation des éléments audio
Le but est d'intégrer un lecteur audio, d'ajouter des boutons de lecture/pause, de créer des listes de lecture et de personnaliser l'apparence de la page web de musique.

Technologies
  • HTML, CSS
  • Javascript
